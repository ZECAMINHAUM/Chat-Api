## Chat-NR

a simple real-time chat, I hope u like it. ( I will deploy soon )

---


First things first! Install dependencies:

```console
$ yarn
```
or
```console
$ npm install
```

Now u can run!

```console
$ yarn dev
$ yarn build
```

#### What I'm using in this project:

    - Socket.io
